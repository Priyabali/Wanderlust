**Wanderlust** is a full-stack web application that allows users to explore, create, and review travel listings. It provides a platform where users can:

* Browse travel destinations and view listing details
* Add new listings with images
* Upload images via Cloudinary
* Leave reviews and ratings for listings
* Register and login securely

The application is built with **Node.js, Express.js, and MongoDB** and follows the **MVC architecture**. It is designed for a smooth and interactive user experience with full CRUD functionality.
